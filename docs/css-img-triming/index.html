CSSで画像を3:2や16:9、1:1にトリミングする（レスポンシブ対応）

 Posted on 2021-07-31

異なる大きさの画像を複数枚並べたいときなどに書くHTMLとCSSのメモ

画像を3:2にトリミング

HTML

<div class="img">
    <img src="/hoge.jpg">
</div>

CSS

.img {
    padding: 0;
    position: relative;
    height: 0;
    padding-bottom: 67%;
    overflow: hidden;
}

.img img {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    width: auto;
    height: auto;
    max-width: 150%; /* トリミング時の倍率 */
    max-height: auto;
}
