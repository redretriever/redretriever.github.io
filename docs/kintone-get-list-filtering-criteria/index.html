kintoneの一覧で設定されている絞り込み条件をJavaScriptで取得する

 Posted on 2021-08-11

kintoneで現在表示しているレコード一覧の絞り込み条件をjavascriptで取得する方法メモ。 クエリ文字列で取得できるので、APIを使ってレコードを取得する場合のクエリなどに使える。 絞り込み条件が指定されていない場合は、nullが返ってくる。

JavaScript

console.log(kintone.app.getQueryCondition());
