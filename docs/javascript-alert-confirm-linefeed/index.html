Javascriptのalert、confirmで改行をする

 Posted on 2021-09-30

JavaScriptのalert、confirmで表示する文言に改行を入れたい場合、改行したい箇所に¥nをいれると改行される

JavaScript

alert('1行目¥n2行目');

confirm('1行目¥n2行目');
