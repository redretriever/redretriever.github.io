WordPressでアイキャッチ画像を有効にしたい

 Posted on 2022-01-14

WordPressのオリジナルテーマ開発時にアイキャッチ画像選択の機能を有効化する実装方法メモ。 有効化することで投稿画面でアイキャッチ画像の選択が可能になる。

PHP

// 各テーマのfunctions.php
add_theme_support( 'post-thumbnails' );
